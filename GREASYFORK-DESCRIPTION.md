# Greasyfork Description

Import your entire Fortnite locker to fortnite.gg with ONE click. Free, open source (MIT), no downloads, no .exe, no password sharing.

WHAT IT DOES
Imports all your cosmetics (skins, pickaxes, emotes, wraps, banners, instruments, cars, LEGO kits and more) to your fortnite.gg locker, sorted by type and rarity. Shows you how many items are new since your last import. Works with lockers of 8000+ items.

Also includes: an account stats panel (V-Bucks, wins, season history and more), a collection breakdown by rarity and type, a selective import filter, saveable filter presets, a clickable chapter filter, a background new-item check, and wishlist tools (how many you own, clear owned in one click, highlights on your wishlist page, and a shop alert when a wishlisted item is in the shop).

IS THIS SAFE?
Yes. You login on Epic's OFFICIAL website (Device Code flow), the script never sees your password. On your Epic account the script only reads data; the single Epic write action is setting the creator code, and only if you explicitly choose it. It cannot buy anything. The token expires after about 2 hours and is cleared automatically after the import (default, can be turned off in settings). No data collection. Every line of code is readable. Do not take our word for it: paste the script into any AI and ask it to audit the code.

HOW TO USE
1. Install this script
2. Open fortnite.gg and follow the panel on the right
3. Connect Epic Account, login, click Import Locker. Done!

A guided tour walks you through your first import. Available in 13 languages, auto-detected from the page.

NOTE
Desktop browsers only (Chrome, Firefox, Edge). Mobile browsers do not support userscripts.

PROBLEMS?
Open the debug console (bug tab), run diagnostics, copy the report and post it on GitHub: https://github.com/ItsReepze/fngg-locker-importer/issues
